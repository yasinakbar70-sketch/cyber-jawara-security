# Plugin Update Checker

Plugin Update Checker adalah library open source untuk menambahkan fitur auto-update plugin WordPress dari GitHub, Bitbucket, atau GitLab. Library ini digunakan oleh banyak plugin premium dan open source.

- Repo asli: https://github.com/YahnisElsts/plugin-update-checker
- Dokumentasi: https://github.com/YahnisElsts/plugin-update-checker/blob/master/README.md

Cara update library ini: download versi terbaru dari repo di atas dan replace file di folder ini.
